**Vibzyy – Full Stack Music Player**

Vibzyy is a **full-stack music player web application** built using the **MERN stack (MongoDB, Express.js, React, Node.js)**.
The application allows users to stream music, manage playlists, and interact with songs through a modern web interface.

This project was developed as part of my internship at **Emertxe WebStack Academy** and helped me gain hands-on experience in **full-stack development, REST API design, authentication systems, and state management**.


## 🚀 Features

* Secure **User Authentication (JWT)**
* Sign up and login functionality
* Browse and play songs
* Create playlists using tags
* Add or remove favourite songs
* Music playback controls:

  * Play
  * Pause
  * Next / Previous
  * Shuffle
  * Loop
* Volume and playback speed control
* Password reset via email
* Edit and update user profile
* Upload profile image


## 🛠 Tech Stack

### Frontend

* React (Vite)
* Redux Toolkit
* Axios
* CSS
* React Icons

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Nodemailer (for password reset emails)
* ImageKit (for profile image upload)

## ⚙ Installation

### Clone the repository

```
git clone https://github.com/nireeksha20/mern-music-player.git
```

### Install backend dependencies

```
cd backend
npm install
```

### Install frontend dependencies

```
cd ../frontend
npm install
```

### Run backend server

```
npm run dev
```

### Run frontend server

```
npm run dev
```

The application should now be running locally.
