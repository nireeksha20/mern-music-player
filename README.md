Vibzyy – Full Stack Music Player

Vibzyy is a full stack music player web application developed as part of my internship at **Emertxe WebStack Academy**.  
The project focuses on building a complete MERN stack application with authentication, music playback, and user interaction features.


Project Overview

The application allows users to:
- Sign up and log in securely
- Browse and play songs
- Create playlists based on tags
- Add and remove favourite songs
- Control music playback (play, pause, next, previous, shuffle, loop)
- Adjust volume and playback speed
- Reset password using email
- Edit user profile details

This project helped me understand full stack development, API integration, authentication, and state management.


Technologies Used

Frontend
- React (Vite)
- Redux Toolkit
- Axios
- CSS
- React Icons

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- ImageKit (for profile image upload)
- Nodemailer (for password reset)
